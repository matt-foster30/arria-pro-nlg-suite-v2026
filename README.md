## Configuration

Settings are grouped by profiles and environment-specific options. A standard setup can include values for integrations, logging, and runtime tuning.

Example layout:

```ini
[profile]
name=default
language=en
audit_logging=true
openai_integration=true
claude_integration=false
```

If your deployment uses separate profile files, keep them in the project configuration directory and update them before launching the toolkit.
